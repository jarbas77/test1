# React App with TypeScript

This project is a React application built with TypeScript, showcasing CRUD operations for managing posts and comments using JSONPlaceholder API. It includes pagination for both posts and comments lists.

## Features

- List all Posts
- Show all comments for a given post
- Create, update, and delete a comment
- Pagination for posts and comments lists

## Technologies Used

- React
- TypeScript
- React Router DOM
- Tailwind CSS (for styling)

## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

- Node.js (v12 or higher)
- npm or yarn

### Installation

1. Clone the repository:

   ```sh
   git clone 
   cd your-repository

2. Install dependencies:
    npm install
    # or    
    yarn install

3. Running the application
    npm start
    # or
    yarn start

Open http://localhost:3000 to view it in the browser.
